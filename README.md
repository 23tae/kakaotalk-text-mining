# KakaoTalk Text Mining

# Usage

1. 가상환경 설치

```shell
python3 -m venv venv
```

2. 가상환경 활성화

```shell
source ./venv/bin/activate
```

3. 패키지 설치

```shell
python3 -m pip install -r requirements.txt
```

4. pc 카카오톡에서 대화기록을 csv로 저장

5. 실행 (경로명은 csv파일의 경로로 입력)

```shell
python3 main.py <file_path>
```
